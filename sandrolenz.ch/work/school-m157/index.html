Phonebook

A school project for M133, where we built an interface for an employee phonebook.

M157

IT-System-Einführung planen und durchführen

In this module we could choose the project ourselves. The idea of an online phonebook came up very quickly since its a simple thing to build.

We used HTML and CSS for our interface. Getting the data from the database was done with PHP and a bit of JavaScript.
